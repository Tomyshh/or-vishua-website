# אור וישועה - אתר מודרני לישיבה

אתר אינטרנט מודרני ומרשים לישיבת אור וישועה בחיפה, בנוי עם הטכנולוגיות החדישות ביותר.

## 🌟 תכונות

- **עיצוב מודרני ונקי** - ממשק משתמש אינטואיטיבי ומרשים
- **תמיכה מלאה ב-RTL** - מותאם לעברית
- **Responsive Design** - עובד מצוין בכל המכשירים
- **אנימציות חלקות** - חוויית משתמש משופרת עם Framer Motion
- **ביצועים מעולים** - אופטימיזציה מלאה עם Next.js 15
- **תמיכה במצב כהה/בהיר** - עם החלפה אוטומטית
- **נגישות** - תמיכה מלאה בתקני נגישות

## 🚀 טכנולוגיות

- **Next.js 15** - Framework React מתקדם
- **TypeScript** - Type safety ופיתוח יעיל
- **Tailwind CSS** - Styling מודרני ויעיל
- **Framer Motion** - אנימציות חלקות
- **Radix UI** - רכיבי UI נגישים
- **Lucide Icons** - אייקונים מודרניים

## 📁 מבנה הפרויקט

```
src/
├── app/                    # Pages and layouts
│   ├── layout.tsx         # Root layout
│   ├── page.tsx           # Home page
│   ├── about/             # About page
│   └── contact/           # Contact page
├── components/            # React components
│   ├── ui/               # UI components
│   ├── sections/         # Page sections
│   ├── navigation/       # Navigation components
│   └── footer/           # Footer component
├── lib/                   # Utilities
└── providers/            # Context providers
```

## 🛠️ התקנה והרצה

1. **התקנת Dependencies:**
```bash
npm install
```

2. **הרצה במצב פיתוח:**
```bash
npm run dev
```

3. **בנייה לייצור:**
```bash
npm run build
```

4. **הרצת האתר בייצור:**
```bash
npm start
```
